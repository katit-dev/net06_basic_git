Các bước đẩy code lên github

Bước 1: Tạo repository trên GitHub

Bước 2: Clone repository về máy tính

Bước 3: Code trên máy tính: tao file, sửa file, xóa file

Bước 4: Chọn những file cần đẩy lên github - đẩy hết: git add . - kiểm tra trạng thái: git status

Bước 5: Đẩy code lên github - git commit -m "Nội dung commit"

Bước 6: Đẩy code lên github - git push origin main
